# India CPI & Air Quality Analysis

## Overview
This project focuses on analyzing Indian Consumer Price Index (CPI) and air quality index data using Python and Jupyter Notebooks. It utilizes open datasets to explore statistical trends, visualize changes over time, and develop actionable insights.

## Contents
- `All_India_Index_july2019_20Aug2020.csv`: Dataset containing India's CPI indices from July 2019 to August 2020.
- `CPI_TimeSeries_Analysis_Satyam_kumar.ipynb`: Jupyter notebook for in-depth CPI time series analysis, including data preprocessing, visualization, and basic forecasting.
- `README.md`: Project documentation.

## Getting Started

1. **Clone the repository:** https://github.com/satyamkumarcode/air_quality_cpi_analysis.git
2. **Open the notebook:**
- Use Jupyter or Google Colab.
3. **Install required libraries:**
- pandas, matplotlib, numpy (see notebook for details).
